## Getting Started

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/Abdullah-Baghraib/x_o.git
   ```
2. Navigate to the project folder:
   ```bash
   cd x_o-flutter
   ```
3. Run the app:
   ```bash
   flutter pub get
   flutter run
   ```
